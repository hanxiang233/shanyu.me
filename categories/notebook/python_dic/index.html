python数据类型 字典

字典的值是以大括号引用的，如，dic = {"a":123, a=456} a可以是一个变量类型，花括号内使用引号引用则是一个字串符

字典是python中唯一映射类型（哈希表）

字典对象是可变的，但是字典的键必须使用不可变对象，一个字典中可以使用不同类型的键值

字典包含一个key对应一个value，key值是字典里唯一并且不可变，value对应key且可以被改变;

字典的方法

keys()

values()

items()

创建一个普通的字典集合的人员信息 

>>> miku = {'name':'miku','age':'16'}
>>> print(miku['name'])
miku
>>> print(miku['age'])
16

访问字典的键值，添加指定的句子

print出的句子如果想要显示的更规整，需要在段落前加空格 

>>> print ("miku age is " + (miku['age']) + " years")
miku age is 16years

向字典里添加信息

字典中添加miku的身高和体重信息，更新增加字典的key和values值 

>>> miku['height'] = 155
>>> miku['weight'] = 46
>>> print(miku)
{'name': 'miku', 'age': '16', 'height': 155, 'weight': 46}

为Luna同学添加个人信息

添加一个空字典，并向空字典里逐渐增加key和values值 

>>> KaguyaLuna = {}
>>> KaguyaLuna['name'] = 'Luna'
>>> KaguyaLuna['age'] = '19'
>>> print(KaguyaLuna)
{'name': 'Luna', 'age': '19'}

修改字典中的值，由于key是不可变的，所以字典只有values值能做改变

修改字典的values就是重新指定key的值即可，这里将Luna的年龄修正为17 

打印字典原来的key和values值
>>> print(KaguyaLuna)
{'name': 'Luna', 'age': '19'}

这样指定的会把name这个key值键顶替掉，也就是说更改后名字为name的key键值会"消失"，这个等同于重新定义了名为KagyaLuna的字典
>>> KagyaLuna = {'age': '17'}
>>> print("The Luna age is " + KagyaLuna['age'])
The Luna age is 17

被重新定义的KagyaLuna的字典
>>> KagyaLuna = {'age': '17'}
>>> print(KagyaLuna)
{'age': '17'}

给Luna再次重新修改年龄，修改指定age这个key对年龄数值进行修改，在定义字典数值时，数值需要使用单引号括起来定义 

打印原Luna的年龄信息
>>> print("The Luna age is " + KagyaLuna['age'])
The Luna age is 17

先重新把KagyaLuna字典定义下(年龄定义为18)，然后再修改Luna年龄
>>> KagyaLuna = {'name': 'Luna', 'age':'18'}

修改Luna的年龄values值，只针对age这个key对应的values做修改，并打印出Luna的信息

Luna age is new 19.

KagyaLuna的字典内容
>>> print(KagyaLuna)

打印介绍Luna同学的句子，py格式的字符和变量之间的空格都是在字符后面或前面加入一个空格符的，+号表示连接两个字符段
>>> print ("This classmate is " + KagyaLuna['name']+ "," "She is "+  KagyaLuna['age']+ " years old" )

删除字典中的一个key和values

比如这里拿miku同学的信息做示范，去除掉miku的年龄key记录，删除的键值对会永久的“消失”

遍历字典键值，首先需要将字典中的key和value赋予到一个指定的字符上，然后使用for循环print来遍历出这个输出内容。例如如下：将人员名称和使用的开发语言关联并遍历输出

字典练习

使用for循环的方式遍历出输入内容

创建一个交互输入的信息记录字典脚本

将输入信息保存至变量，然后将变量赋值给字典(赋值过程中定义key，value则是交互时输入的)。再使用for循环定义print出的k值和v值，k和v去引用字典中的key和value，字典使用ltems输出key和value元素，元素由k、y捕获并被print出来

使用%s引用k和y的输出，%号则是去引用k和y元素，输出结果会将字典按照key: value输出 

#!/usr/bin/python
miku = {}         创建一个空字典
name = raw_input("Please input name: ")     执行时对用户提出交互输入
age = raw_input("Please input age: ")
gender = raw_input("Please input gender: ")
miku['name'] = name          把交互输入应用到变量(字典对应的key中)
miku['age'] = age
miku['gender'] = gender
for k, v in miku.items():         for循环字典的值,将字典以k(key)，v(value)形式列出
    print "%s: %s" % (k, v)         print遍历显示字典数值，%s表示对k和v的引用，使用双引号扩起
print 'end'                         被引用的k和v以括号引起，如果是一个值则不需要使用括号
                                    %s: %s引用两个变量的值，可在变量值中加入分隔符
